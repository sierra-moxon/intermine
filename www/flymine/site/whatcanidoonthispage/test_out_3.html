Upload

FlyMine can operate on lists of data. Lists can then be analysed on a list analysis page, or can be used in (template) queries. The Upload page allows you to create a list from an external source. The list can contain a mix of identifier types (eg symbols, names, organism database id,) but it cannot contain different object types (eg a list of genes and proteins is currently not possible).

View

The View page shows all public lists (name starting with 'PL'), as well as your own (marked with 'my') - if you are logged in. Clicking on a List name will show the List analysis page for this list. We can add new public lists at any time - it only takes a few minutes - so if you have a suggestion, let us know.

You can merge, subtract and find common members if you have more than one list.
